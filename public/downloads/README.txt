Place your Nova Browser APK file here as "nova-browser.apk"

The website will link to downloads/nova-browser.apk for users to download the app.

To generate the APK:
1. Run: flutter build apk --release
2. Copy the generated APK from build/app/outputs/flutter-apk/app-release.apk
3. Rename it to nova-browser.apk
4. Place it in this downloads folder
